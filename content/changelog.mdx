# Changelog

All notable changes to SimpleNS will be documented here. For detailed release notes, visit our [GitHub Releases](https://github.com/SimpleNotificationSystem/simplens-core/releases).

## Release Structure

Each release entry contains:
- **version**: Semantic version number
- **date**: Release date  
- **tag**: Optional label (e.g., "Latest")
- **description**: Brief summary
- **dockerImages**: Docker image tags
- **features**: Grouped feature highlights
- **bugFixes**: List of bug fixes (optional)
